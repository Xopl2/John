# John
An unpredictable AI chatbot with rotating unhinged personalities.
Built with React, Node.js/Express, and Ollama (local LLM).

## Tech Stack
- Frontend: React
- Backend: Node.js + Express
- AI: Ollama (Llama 3.1 8B)
